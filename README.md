# Installation
1. Clone the [starter code](https://github.com/coding-boot-camp/fantastic-umbrella)
2. Run `npm install`
3. Run `node seed/index.js`
4. Run `npm start`
5. Go to the Insomnia application to test the routes using the `3001` port
